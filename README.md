## A repository dedicated to compare sorting methods in different langagues

## File structure

In order to add to the implementations it is expected to follow this structure.
```
.
├─ Java
│  ├── Impl 1
│  └── Impl 2
│
├── Rust
│   ├── workspace
│   │   ├── Impl 1
│   │   └── impl 2
│   └── Cargo.toml
│
├── Run.ps1
├── Run.sh
└── Makefile
```
